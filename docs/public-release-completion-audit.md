## Current stopping conditions

Build 60 cannot be called a fully verified public release candidate until Apple processing, the consolidated iPhone test and explicit UI freeze, performance/stability evidence, exact Devin-account retention confirmation, App Store privacy publication, Developer ID/notarization and clean-account Connector checks, private review credential, final screenshots, and explicit release approval are all recorded. Build-spec targets such as the seven-day crash-free window and timed OTA rollback drill remain real gates unless the specification is explicitly changed.
